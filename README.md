# CoronaSummary
This is a open source app for corona status in the world

Features Included: 
1. MVVM Clean Arch
2. SOLID Principles
3. Koin Dependency
4. Motion Layout
5. Jetpack Components
6. Coroutines and Flow
8. Retrofit and Room
9. Orientation Handeling
10. mockk unit testing (Pending)
